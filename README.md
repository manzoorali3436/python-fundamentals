# python-fundamentals
A collection of my Python fundamentals and hands-on practice while transitioning into Python and AI development.

This repository covers core Python concepts with clean, organized examples, including variables, data types, operators, conditions, loops, functions, OOP, file handling, exceptions, modules, and more.

The goal of this repository is to build a strong Python foundation before moving on to automation, data analysis, machine learning, AI, and real-world projects.
